# Password Strength Checker (Python)

This project is a Python-based password strength checker that evaluates passwords based on:

- Length (minimum 8 characters)
- Uppercase and lowercase letters
- Numbers
- Special characters

## Features
- Classifies passwords as Weak, Moderate, or Strong
- Provides feedback on how to improve password strength
- Demonstrates basic cybersecurity principles such as secure password practices

## How to Run
1. Run the Python script
2. Enter a password when prompted
3. View strength classification and suggestions

## Technologies Used
- Python
